罗振宇:时间的台阶

命题:怎样做一个更有价值的人
罗振宇的比喻:人生是”一段时间”
衡量价值的参考不应当是钱 而是”墓志铭” 是时间

  • 台阶1:把时间用好:
    • 财力允许的情况下尽量离单位近一点
    • 时间的花费是始终不能适应的
  • 台阶2:同一段时间的受益尽可能多元化:魅力?
    • 最有趣的人最有上进心的人都愿意去的人 => 明星公司
    • 选择的维度:不应是钱
    • 时间花进去了 看不到的受益心里得有:长本事 长见识 和可能未来影响你一生的人协作
  • 台阶3:同一笔时间获得更多价值
    • 扩散价值: 利用媒体 (出书 创作 最终扩散)
    • 得会表达 得会输出 写文章做演讲做分享
    • 狭窄领域的扩散方法: 葛文德大夫: 总结分享领域的行为方式和认知框架 => <<清单革命>> => 扩散本格行业的优秀方法(独特的认知模型)到大众视野
  • 台阶4:[难了]找未来的时间要价值:把自己当做上市公司 –> 靠谱
    • 事事有回音 凡是有交代 件件有着落 -> 回到品质
    • 给你身边牛逼的人一个确定的预期:靠谱是第一步
  • 台阶5:[罗振宇的最高台阶]买别人的时间再卖掉 : 投资
    • 不信任是阻碍:
      • 生活在质疑中是低效的 : 嗨的确是很多很多事情
      • 选择信任他人 : 掏钱买服务的算法 : 花钱买的是大量人的时间
      • 经营自己的方式 -> 买别人的时间,画好自己时间的生意
  • 前三个台阶是战术:乘法: 百分之20 * 百分之20 = 百分之4 跨界的方法
  • 后两个台阶是战略:基数:
