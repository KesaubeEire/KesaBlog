列表表达式

#基本语法：[expr for iter_var in iterable]

#原型:
>>> L= []
>>> for x in range(1 , 10):
...     L.append(x*x)
... 
>>> L
[1, 4, 9, 16, 25, 36, 49, 64, 81]

#简化:
>>> print [x*x for x in range(1 , 11)]
[1, 4, 9, 16, 25, 36, 49, 64, 81, 100]
>>>

总结:
相当于把要 append的元素条件放在了[ ] 后的第一个位置
输出的仍然是一个列表

Zip:

zip 返回一个将多个可迭代对象组合成一个元组序列的迭代器。
每个元组都包含所有可迭代对象中该位置的元素。
zip封装:

list = zip(['a', 'b', 'c'], [1, 2, 3])
>>>list = [('a', 1), ('b', 2), ('c', 3)]

另一种玩法
x_coord = [23, 53, 2, -12, 95, 103, 14, -5]
y_coord = [677, 233, 405, 433, 905, 376, 432, 445]
z_coord = [4, 16, -6, -42, 3, -6, 23, -1]
labels = ["F", "J", "A", "Q", "Y", "B", "W", "X"]

points = []
### 注意这里的玩法
for label, x, y, z in zip(labels, x_coord, y_coord, z_coord):
    points.append("{}: {}, {}, {}".format(label,x,y,z))

for point in points:
    print(point)

变量命名

1. 只能在变量名称中使用常规字母、数字和下划线。不能包含空格，并且需要以字母或下划线开头。
2. 不能使用保留字(preserved words)或内置标识符(build-in identifier)
3. 在 python 中，变量名称的命名方式是全部使用小写字母，并用下划线区分单词。
4. 我们命名变量的方式称之为 snake case，因为我们用下划线连接单词 my_variable

#列表的一些实用函数:
len() :   返回列表中的元素数量。
max() :
  返回列表中的最大元素。最大元素的判断依据是列表中的对象类型。
  数字列表中的最大元素是最大的数字。字符串列表中的最大元素是按照字母顺序排序时排在最后一位的元素。
  因为 max() 函数的定义依据是大于比较运算符。如果列表包含不同的无法比较类型的元素，则 max() 的结果是 undefined。
min() :   返回列表中的最小元素。它是 max() 函数的对立面，返回列表中的最小元素。
sorted(): 返回一个从最小到最大排序的列表副本，并使原始列表保持不变。
replace():替换旧的换成新的
