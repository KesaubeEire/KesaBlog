了解各种节点(Node)

这里强烈推荐洪流学堂的
ShaderGraph节点大全
以下为一些自我的心得
Lerp

插值变换
输入A B的值
以及输入插值T
当T为0时       输出A
当T为1时       输出B
当T为0-1时     输出相应的中间插值
当T不属于0/1时  输出相应的超出值
挺好理解的

Remap

将输入值的最大最小映射到另一个最大最小
输入值低于要求最小默认为最小
输入值高于于要求最大默认为最大

Saturate

相当于简化版的Remap
只不过对所有范围
都映射到(0,1)

Step

将Edge的输入值
用In作为阈值
高于阈值的作为White
低于阈值的作为Black

Returns 1 if the value of input In is greater than or equal to the value of input Edge, otherwise returns 0.

Fresnel Effect[^1]

使得材质可以因为角度不同看到不同的颜色
原理是菲涅尔效应的
理想的示例:
  - 水面
  - 各种发光球和罩

经常和Lerp配合
因为规定颜色最理想的是使用Lerp
规定主题色和边缘色

The Fresnel Effect node approximates this by calculating the angle between the surface normal and the view direction. The wider this angle is, the greater the return value will be.

了解总输出(Master)

PBR 基于物理渲染

Physically Based Renderer
    - metalic
        - 灰度,白的高光越来越弱
    - specular
        - 模拟材质很好:金属电镀

Position

    - 通过使用一些特殊的贴图或者算法控制模型上的顶点

通道系列 : Tables from Unity Shader Wiki

PBR Master Ports

Name Type Binding Description
Albedo V 3 None Defines material’s albedo value. Expected range 0 - 1.
Normal V 3 Tangent Space Normal Defines material’s normal value. Expects normals in tangent space.
Emission V 3 None Defines material’s emission color value. Expects positive values.
Metallic V 1 None Defines material’s metallic value where 0 is non-metallic and 1 is metallic. Only available in Metallic Workflow mode.
Specular V 3 None Defines material’s specular color value. Expected range 0 - 1. Only available in Specular Workflow mode.
Smoothness V 1 None Defines material’s smoothness value. Expected range 0 - 1.
Occlusion V 1 None Defines material’s ambient occlusion value. Expected range 0 - 1.
Alpha V 1 None Defines material’s alpha value. Used for transparency and/or alpha clip. Expected range 0 - 1.
Alpha Clip Threshold V 1 None Fragments with an alpha below this value will be discarded. Requires a node connection. Expected range 0 - 1.


Unlit Master Ports

Name Type Description
Color V3 Defines material’s color value. Expected range 0 - 1.
Alpha V1 Defines material’s alpha value.
Used for transparency and/or alpha clip.
Expected range 0 - 1.
Alpha Clip Threshold V1 Fragments with an alpha below this value will be discarded. Requires a node connection. Expected range 0 - 1.


Master Parameters

Name Type Options Description
Workflow Dropdown Metallic, Specular Defines workflow mode for the material
Surface Dropdown Opaque, Transparent Defines if the material is transparent
Blend Dropdown Alpha, Premultiply, Additive, Multiply Defines blend mode of a transparent material
Two Sided Toggle True, False If true both front and back faces of the mesh are rendered



- Albedo:漫反射

    - 物体的固有色(Like Diffuse)
    - 这里的颜色算法是RGB
  • Normal:法线
  - 用来计算光照的反射,在低面模上模拟高面模的效果
    - 应用:无法线效果:纪念碑谷类的素动漫风格
  • Emission:自发光
    - 不受到其他光照的影响
    - 模型和uv对应的
  • Metallic:金属度
    • Smoothness:度
    - 和Specular二选一不共存
    - 反光度的样子
  • Specular:反射度(英文是镜子的意思)
    • Smoothness:度
    - 和Metallic二选一不共存
    - 另一种反光度的样子


上图是一个各种物体的反光度的一个展示
  • (Ambient) Occlusion
    - 环境闭塞:各种遮盖阴影的效果,表现真实性,
    - 用灯光去解决这种问题的开销是很大的,这样节省计算资源
  • Alpha
    • AlphaClip Threshold 
    - Alpha里的算法...好像是有点不一样
大把时间迷茫 几个瞬间成长
