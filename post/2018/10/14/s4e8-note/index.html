笔记|奇葩说S4E8_Note

Kesa / 2018-10-14

Topic

如果有个按钮能看到伴侣有多爱你
你要不要按？

正1:傅首尔

要按:

  1. 按钮就是用来按的
  2. 既然爱情是必须要被证明的,何不干脆清清楚楚地证明它
  3. 让老公feel惭愧,
    分比我低:别输给我,
    分比我高:还对你这么好,说明我对你负责的很
  4. 实质:愿不愿意面对量化的爱情
    数值无论多少 我愿意就好

反1:沈玉琳

不按:

  1. 己所不欲 勿施于人
  2. 有摄像机介入的真人秀就不是真实的
  3. 过分依赖机器容易使人丧失爱人的本能
  4. 感情世界够纷扰,别为自己多添烦恼

正2:颜如晶

要按:

  1. 钮的分数是可以变的:重点在下一次 -> 给人改变的动力
  2. 这只是一个参考: 爱的分数和爱的表现方式是两回事
    多一个参考为啥不用
  3. 作为一个工具是很好的: 类比 啪啪钮 是鼓励 -> 拉票

反2:杨奇函

不按:

  1. 有考试就容易出现套路
  2. 测试爱人的本质就是测试人性 : 人性太脆弱
    大家不是圣人:人类有智慧的第一件事情是出卖同类
    我们都不完美 -> 需要给彼此空间

正3:马薇薇

要按:

  1. 反驳:1.没有人性的测试 2.爱情可以量化->看科技的变化
  2. 这个案件是”沟通”的按键
    - -> 按钮帮助我们更好的沟通
    - - - -> 帮助我们更健康地对待爱情
  3. 吐槽:整体感觉不够好,不是她的正常水平

反3:邱晨

不按:

  1. 反驳 : 其实没有降低沟通成本
  2. 常见的三连击:你爱不爱我 有多爱我 很爱是多爱
    - -> 让沟通成本升级
    - - - -> 沟通没有捷径 只有不偷懒 按钮给了我们虚幻的错觉
    - - - - - -> 不要轻易证明自己
  3. 程度不重要 方式才重要
    数值不重要 体验才重要
    真实可能没那么重要 真实感才重要
  4. 爱是什么不知道,但是爱一定不是股票
    我认为爱是一段故事 不要把它拆碎了
    用一个一个字去衡量它的价值

正4:黄执中

要按:

  1. 真实不会伤害人 会伤害人的是对真实的解释
    -> 没有人伤害你 只有自己伤害自己
    - -> 什么叫恨:我对么想伤害你
    - -> 什么叫爱:我能让你伤害我到什么程度

  2. 辩题讨论的核心:我们要不要面对真实真相
    -> 好的解释让你愿意知道真相
    -> 坏的解释让你想要逃避真相
    - - -> 终极的逃避真相:你为什么让我知道

  3. 真实无法逃避
    逃避真相会付出代价
    -> 如果你保护自己的唯一方式是我不知道别人有多爱我
    -> 那你幸福的权力在别人的手上->你丧失了对这件事的解释权
    - -> 所有的这些东西都会被发明出来
    - -> 我们每个人面对的一切随着科技的进步都在被揭示
    - - -> 你现在不付出面对真相的代价
    - - -> 你将来得付出不愿意面对真相的代价
    练习面对真相的能力 而不是逃避
    - >给自己一个好的解释

4.结辨:
我们从来不是潇洒的人
我们没有想象中那么潇洒
好奇心不会因为这种XJB东西消失
浪漫是我们面对一切无能为力和未知的时候最安慰人的方式
反正我不知道答案:我误以为不知道答案才是浪漫
另一种浪漫我们只是没有体会到
才能让我们向着更深远的 更核心的浪漫前进
这只不过是我们现在想象不到的好奇心而已

反4:肖骁

要按:

知道你们要上价值上的很可怕
要把你们拉回地面

各个反驳
- -> 反1: 购物体验 比 购什么物 重要
- -> 反2: 耍赖过去
- -> 反3: 天天撒娇就是沟通吖 降低沟通成本就给吵架加风险
- -> 反4: 拒绝接触丑陋的真相 分开不是因为真相是因为感觉

  1. 这个按钮是个科技:风险大 效果差 易上瘾
    - -> 让我们失去对爱情的好奇心

  2. 这个辩题是把爱情放进应试教育
    - -> 挂科到最后相当于成长 : 不断从失败中累积经验
    - -> 按钮到最后相当于作弊 : 你什么都没学到

  3. 这个真相:你以为你服务了伴侣 其实是伤害了自己
    - -> 不要轻易去窥探爱情的真相:会让自己找不到自己
    - ->
    - -> 拿自己举例
    - -> 不再真实了 为了赢 为了按键改变自己
    - -> 连自己都不再喜欢自己了
    - -> 让自己变得尖锐犀利 不在乎朋友们了

  4. 按按钮是担心对方不够爱自己
    - -> 相当于把爱自己的权利交给了别人
    - -> 不输入你伴侣的名字
    - -> 输入你自己的名字

  5. 按下去:是已知的真相
    不按他:是无穷的想象

用坦诚和真实
把真实的焦虑和命去拼一定是ok的

高晓松 : 爱情最后从爱自己的角度写是最好的
大把时间迷茫 几个瞬间成长
